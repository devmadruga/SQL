Documento de modelagem entregue pelos Analistas de Sistemas:

Uma farmácia comercializava vários tipos de produtos (remédios, cosméticos, etc). Além do tipo, para cada produto são registrados o código, designação, composição, fabricança e preço de venda. Quando um cliente compra pela primeira vez na farmácia, são registrados os seus dados (nome, endereço, telefone, código postal, localidade e cpf). Depois de cadastrado, sempre que um cliente compra um determinado produto é registrado seu código de cliente, os produtos adquiridos, o nome do médico (no caso das receitas médicas), bem como o total da venda. Cada compra é identificada por um número para facilitar as consultas à base de dados.
